🎾 TennisIQ
Transforming Tennis Performance Through Data

TennisIQ is an analytics platform built to help junior tennis players identify performance patterns, uncover weaknesses, and receive actionable coaching insights using match statistics.

As a nationally and internationally ranked junior player, I experienced firsthand how difficult it was to objectively analyze performance after matches. TennisIQ was created to bridge that gap.

Features
📊 Interactive performance dashboard
📈 Match trend visualization
🎯 TennisIQ performance scoring system
🧠 AI-generated coaching recommendations
📂 CSV match upload support
⚡ Built using Next.js and React
Why I Built This

Competitive junior tennis often lacks affordable performance analytics tools.

TennisIQ aims to democratize data-driven coaching by providing players with meaningful insights using statistics they already track.
